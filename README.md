# 🔭 DB Visualizer
 
Upload a `.sql` file and instantly visualize your database schema — tables, columns, PKs, FKs, and relationships — as an interactive graph.
 
---
 
## Tech Stack
 
| Layer | Tech |
|---|---|
| Backend | Spring Boot 3 + JSQLParser |
| Frontend | React + React Flow (Vite) |
| Deploy | Docker + Render / Railway + Vercel |
 
---
